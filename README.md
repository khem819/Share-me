# Share Me 🚀

A full-stack web application built for sharing and managing content/images in a simple and user-friendly way.

## ✨ Features
- User authentication
- Upload and share posts
- Like and interact with content
- Responsive UI
- Fast and modern frontend

## 🛠 Tech Stack
- React
- Vite
- Node.js
- Express
- MongoDB
- Tailwind CSS

## 📦 Installation

```bash
git clone https://github.com/khem819/Share-me.git
cd Share-me
npm install
npm run dev
